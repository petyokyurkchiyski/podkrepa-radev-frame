# 🎯 Генератор на профилни снимки с рамка

Автоматичен инструмент за добавяне на рамка към профилни снимки за подкрепа на Румен Радев.

## 🚀 Инструкции за стартиране

### 1. Инсталиране на зависимостите

Отворете терминал в папката на проекта и изпълнете:

```bash
npm install
```

### 2. Стартиране на сървъра

```bash
npm start
```

Сървърът ще стартира на `http://localhost:3000`

### 3. Отваряне в браузър

Отворете браузъра и отидете на:
```
http://localhost:3000
```

## 📸 Как да използвате

1. **Качете снимката си** - Кликнете на бутона "Изберете вашата снимка"
2. **Настройте** (по желание):
   - Размер на снимката
   - Позиция (X и Y)
   - Прозрачност
3. **Изтеглете резултата** - Кликнете "Изтегли снимката"

## 🌐 Публикуване онлайн

За да направите инструмента достъпен за всички онлайн, можете да го качите на:

- **Heroku**: Безплатен хостинг за Node.js приложения
- **Vercel**: Безплатен хостинг с автоматично разгръщане
- **Railway**: Лесно разгръщане на приложения
- **DigitalOcean App Platform**: Професионален хостинг

### Пример за Heroku:

1. Създайте `Procfile` файл:
```
web: node server.js
```

2. Инсталирайте Heroku CLI и изпълнете:
```bash
heroku create
git push heroku main
```
